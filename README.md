# Веб-приложение для Обработки Изображений

## Описание

Это веб-приложение позволяет пользователям загружать изображения и применять различные методы обработки для улучшения или анализа изображений. Приложение реализует следующие методы:

1. **Низкочастотные фильтры (сглаживающие)**
   - Средний фильтр

2. **Локальная пороговая обработка**
   - Метод Отсу
   - Адаптивное среднее

3. **Адаптивная пороговая обработка**
   - Метод Бенсен

Дополнительно реализована функциональность увеличения (приближения) изображений при клике на них для детального просмотра.

## Основные возможности

- **Загрузка изображений:** Поддержка различных форматов изображений через стандартный файловый ввод.
- **Отображение изображений:** Показывает оригинальное и обработанное изображение рядом для удобного сравнения.
- **Применение фильтров:** Возможность применения различных методов обработки изображений с одним нажатием кнопки.
- **Увеличение изображений:** При клике на любое из изображений (оригинальное или обработанное) открывается модальное окно с увеличенной версией.
- **Минималистичный интерфейс:** Чистый и интуитивно понятный дизайн, удобный для пользователей всех уровней.

## Принципы Работы

### 1. Низкочастотные фильтры (Сглаживающие)

**Средний фильтр:**
Средний фильтр — это простой метод сглаживания изображения, который заменяет каждый пиксель средним значением его соседних пикселей в заданном окне (например, 3x3). Это помогает уменьшить шум и сгладить изображение.

**Алгоритм:**
1. Для каждого пикселя вычисляется среднее значение красного, зелёного и синего каналов его соседей.
2. Пиксель заменяется этими средними значениями.

### 2. Локальная Пороговая Обработка

#### Метод Оцу

Метод Оцу автоматически определяет оптимальный порог для разделения изображения на фон и объект, основываясь на максимизации межклассовой дисперсии.

**Алгоритм:**
1. Вычисляется гистограмма изображения.
2. Для каждого возможного порога вычисляется межклассовая дисперсия.
3. Порог с максимальной межклассовой дисперсией выбирается как оптимальный.

#### Адаптивное Среднее

Этот метод устанавливает порог для каждого пикселя на основе среднего значения яркости в его локальном окне с некоторым коэффициентом.

**Алгоритм:**
1. Для каждого пикселя вычисляется среднее значение яркости в окне (например, 15x15 пикселей) вокруг него.
2. Порог устанавливается как некоторое процентное значение от этого среднего (например, 90%).
3. Пиксель бинаризуется на основе этого порога.

### 3. Адаптивная Пороговая Обработка (Метод Бенсен)

Метод Бенсен учитывает локальный контраст при определении порога. Если контраст в окне ниже заданного порога, пиксель устанавливается как белый; иначе используется локальное среднее для порога.

**Алгоритм:**
1. Для каждого пикселя вычисляется среднее значение и контраст (разница между максимальным и минимальным значениями) в его локальном окне.
2. Если контраст ниже порогового значения, пиксель устанавливается как белый.
3. В противном случае пиксель бинаризуется на основе локального среднего значения.

3. **JavaScript:**
   - Добавлены обработчики событий `click` для `originalCanvas` и `processedCanvas`, которые вызывают функцию `openModal()`.
   - Функция `openModal()` получает данные изображения из `canvas` и отображает их в модальном окне.
   - Функция `closeModal()` закрывает модальное окно.
   - Добавлен обработчик события `window.onclick`, который закрывает модальное окно при клике вне изображения.
